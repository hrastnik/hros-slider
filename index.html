Usage

Include script on your page.

    <script src="hros-slider.min.js"></script>

Initialize slider when your content is loaded.

    <script>
        new hros.Slider(); 
    </script>
  • Wrap everything in .hros-slider.
  • Add .hros-slider-goto to goto buttons and give them 'data-goto' attribute with index of the slide they go to.
  • Add .hros-slider-slide to slides.

Examples:

Using CSS

  • Active slide has the .active class.
  • Inactive slides have the .inactive class.
  • Slides with index less than the index of active slide have the .prev class.
  • Slides with index more than the index of active slide have the .next class.
  • Active goto button has the .active class.
  • Inactive goto buttons have the .inactive class.
  • Goto buttons with index less than the index of active slide have the .prev class.
  • Goto buttons with index more than the index of active slide have the .next class.
  • Container gets data-active attribute set to current active slide index.

Using JavaScript

  • Slide element can have the following attributes: data-prev-to-active, data-active-to-next, data-next-to-active data-active-to-prev, data-prev-to-next, data-next-to-prev.
    The attributes can contain the name of the function that is called when the slide transitions between states.
  • For example: 
            <script>
          function hide(el) { el.style.display = 'none'; }
          function show(el) { el.style.display = 'block'; }
        </script>

        <div class="slider-js-slide" 
          data-prev-to-active="show" 
          data-next-to-active="show" 
          data-active-to-next="hide"
          data-active-to-prev="hide">
            ...
        </div>

Options and defaults:

            var slider = new hros.Slider({
                classPrefix: "hros-slider",      // Class prefix all classes recieve
                activeClass: "active",           // Classname the active slide recieves 
                inactiveClass: "inactive",       // Classname all inactive slides recieve
                prevClass: "prev",               // Classname all slides with index < activeIndex recieve
                nextClass: "next",               // Classname all slides with index > activeIndex recieve
                slideSufix: "slide",             // Slides receive the classname: classPrefix + '-' + slideSufix 
                gotoSufix: "goto",               // Goto buttons receive the classname: classPrefix + '-' + gotoSufix
                arrowLeftSufix: "arrow-left",    // Left arrows receive the classname: classPrefix + '-' + arrowLeftSufix
                arrowRightSufix: "arrow-right"   // Right buttons receive the classname: classPrefix + '-' + arrowRightSufix
            });
